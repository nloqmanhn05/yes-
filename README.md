# yes-✅

A web-based outfit checker built with HTML, CSS, and JavaScript.

---

## 🚀 Getting Started

### Prerequisites
Make sure you have [Git](https://git-scm.com/downloads) installed on your machine.

### Installation

1. **Clone the repository**

   Open your Command Prompt (Windows) or Terminal (macOS/Linux) and run:

   ```bash
   git clone https://github.com/nloqmanhn05/yes-
   ```

2. **Navigate to the project folder**

   ```bash
   cd yes-
   ```

3. **Run the project**

   Open `index.html` in your browser — simply double-click the file, or right-click and choose **Open with > Browser**.
